# Stock Market Analysis Using Vanilla JS

A lightweight, web-based application for analyzing stock market trends using raw HTML, CSS, and Vanilla JavaScript. This project visualizes stock data with interactive charts and allows users to filter data across different timeframes (1 Month, 3 Months, 1 Year, 5 Years).
